# Binary < - > Denary Convertor in Python
### Binary to Denary
```
    user$: python BinaryToDenary.py
```
Example:

Input
```
    111001
```
Output
```
    Binary: 111001
    Denary: 57
```

### Denary to Binary
```
    user$: python DenaryToBinary.py
```
Example:

Input
```
    54
```
Output
```
    Denary: 54
    Binary: 110110
```
